## JP.TKY.IIJ 是什么？这条线路到底"优"在哪

先说品牌。VMISS 是 2022 年成立的加拿大主机商，主打香港、日本、韩国、美国几个对中国大陆访问优化的小众机房，产品节奏不算快，但线路选得挺讲究。日本这条产品线现在分了三个系列：JP.TKY.TRI 是三网优化（电信 CN2、联通 4837、移动 CMIN2，价格贵一倍），JP.TKY.BGP 是 BGP 混合线路（电信走 IIJ、联通走软银 SoftBank、移动走 CMI），而 JP.TKY.IIJ 则是纯 IIJ 单 ISP 路线。

JP.TKY.IIJ 这条线路的核心卖点是**便宜 + 大带宽 + 单 ISP 直连**。三网回程统一走日本 IIJ（Internet Initiative Japan）网络：

- **中国电信**：IIJ 直连，回程为 163 普通线路

- **中国联通**：IIJ 直连，回程为 4837 普通线路

- **中国移动**：CMI 国际直连，回程走 58453/9808

不是高端 CN2 GIA，也不是 9929 这种联通精品网，就是日本本土最大的 ISP 之一 IIJ 的普通直连。换句话说，它不像 TRI 系列那样在晚高峰走专门优化过的路径，但胜在带宽大（500Mbps 起步）、价格压得很低——同档配置比 TRI 便宜一半左右。

VMISS 自己在商品页面上有一段很实在的提示，建议原文贴出来：

> 由于 IIJ 和 SoftBank 路线近年来已成为连接日本最经济实惠的选择，全球用户群显著增加，整体性能逐渐下降。在高峰时段尤其可能遇到丢包增加、延迟变高、速度变慢。如果对稳定性、速度和更高层级的连接有要求，建议选择 JP.TKY.TRI 系列。

官方自己都明说晚高峰可能不稳，这点必须如实告诉你——这不是营销话术，是写在购买页的免责声明。所以买 IIJ 之前，先想清楚你的使用场景对晚高峰敏感不敏感。
